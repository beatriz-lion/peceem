Tutorial: Primeiros Passos no Terminal

O terminal/linha de comando é uma ferramenta poderosa, especialmente no Linux e macOS. Veja alguns comandos básicos:

ls - Lista arquivos no diretório atual

cd pasta - Entra em uma pasta

mkdir nova_pasta - Cria uma nova pasta

rm arquivo - Remove um arquivo

sudo - Executa comandos como administrador

Experimente estes comandos em um terminal Linux ou no PowerShell do Windows.
