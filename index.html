Sobre o Projeto

O projeto PECEEM foi desenvolvido para preencher uma lacuna importante na formação dos jovens, oferecendo conhecimentos fundamentais em tecnologia que serão úteis tanto na vida acadêmica quanto profissional.

Nosso objetivo é:

  • Democratizar o acesso ao conhecimento computacional
  • Preparar os estudantes para os desafios da era digital
  • Desenvolver habilidades de resolução de problemas
  • Estimular a criatividade e o pensamento lógico

O projeto é voltado para estudantes de todas as séries do ensino médio, com abordagem adaptada para cada nível de conhecimento.

Conteúdos Programáticos

Fundamentos da Computação

História e evolução dos computadores, componentes de hardware, sistemas operacionais e conceitos básicos de redes.

Internet e Segurança Digital

Funcionamento da internet, navegação segura, proteção de dados pessoais e identificação de notícias falsas.

Pensamento Computacional

Abstração, decomposição de problemas, reconhecimento de padrões e desenvolvimento de algoritmos.

Ferramentas Digitais

Pacote Office/Google Workspace, editores de texto, planilhas eletrônicas.

Introdução à Programação

Lógica de programação com Python ou JavaScript, estruturas básicas e desenvolvimento de pequenos projetos.
